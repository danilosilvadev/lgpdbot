# Objective
    This project aims to be a dashboard for managing the cookies of user registered domains.

# Instalation
```npm install```

# Run
```ng serve --open```

# Main Technologies
* Angular 8
    ** NgRX
* Firebase
    ** NgrxFirestore
* SCSS

# NgrxFirestore

## API DOCUMENTATION:

All ids are using acronims.

```
    uid: user id.
    cid: cookie id.
    did: domain id.
    gid: group id.
```

## We have 2 endpoints:
```
    /user
    /cookies
```
